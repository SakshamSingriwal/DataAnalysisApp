Clone the Repository\n\n```bash\ngit clone https://github.com/SakshamSingriwal/DataAnalysisApp.git\ncd DataAnalysisApp/DataAnalysisApp\n```\n\n### 2. Create Virtual Environment\n\n```bash\n# Windows\npython -m venv venv\nvenv\\Scripts\\activate\n\n# macOS/Linux\npython3 -m venv venv\nsource venv/bin/activate\n```\n\n### 3. Install in Development Mode\n\n```bash\n# Install with all dependencies\npip install -r requirements.txt\n\n# Install as editable package\npip install -e .\n\n# Install development tools (optional)\npip install pytest pytest-cov black flake8 mypy\n```\n\n### 4. Run the Application\n\n```bash\nstreamlit run app.py\n```\n\n---\n\n## 📁 Project Structure\n\n```\nDataAnalysisApp/\n├── __init__.py                 # Package initialization\n├── config.py layout, navigation, and UI orchestration\n- Manages Streamlit session state\n- Imports and integrates all feature modules\n\n```python\nimport streamlit as st\nfrom file_handler import read_any_file\nfrom preprocessing import run_preprocessing\nfrom ml_models import run_ml_models\n# ... more imports\n```\n\n**config.py**\n- Centralized configuration for all settings\n- Color schemes, paths, thresholds, and constants\n- Feature flags and environment detection\n- Easy to customize without modifying code\n\n**file_handler.py**\n- Universal file reading for 20+ formats\n- Automatic encoding detection\n- Error handling and metadata extraction\n- Returns: `(DataFrame, metadata_dict, error_code)`\n\n**utils.py**\n- Reusable utility functions\n- Custom Streamlit UI components (info/success/warning boxes)\n- Data quality scoring and formatting helpers\n\n### Feature Modules\n\n**preprocessing.py**\n- Data encoding (One-Hot, Ordinal, Label)\n- Feature scaling (StandardScaler, MinMaxScaler, etc.)\n- Missing value imputation (Simple, KNN)\n- Feature engineering tools\n\n**ml_models.py**\n- Classification models (12+ algorithms)\n- Regression models (10+ algorithms)\n- Clustering algorithms (5+ variants)\n- Model evaluation and comparison\n- Hyperparameter tuning\n\n**deep_learning.py**\n- Neural network architecture builder\n- Dense layers with activations\n- Dropout and batch normalization\n- Training with early stopping\n- Performance visualization\n\n**stats_tests.py**\n- Comprehensive hypothesis testing\n- Parametric tests (t-tests, ANOVA)\n- Non-parametric tests (Mann-Whitney, Kruskal-Wallis)\n- Correlation and proportion tests\n- P-value interpretation\n\n**merge_helpers.py**\n- Multi-file merging utilities\n- Join operations (inner, outer, left, right)\n- Data validation and conflict resolution\n\n---\n\n## 🔄 Data Flow\n\n```\n1. File Upload (file_handler.py)\n   ↓\n2. Data Validation (utils.py: compute_quality)\n   ↓\n3. Exploratory Analysis (app.py)\n   ↓\n4. Data Cleaning (preprocessing.py)\n   ↓\n5. Feature Engineering (preprocessing.py)\n   ↓\n6. Statistical Analysis (stats_tests.py)\n   ↓\n7. Model Training (ml_models.py OR deep_learning.py)\n   ↓\n8. Predictions & Export (app.py)\n```\n\n### Session State Management\n\nStreamlit session state is used to maintain:\n\n```python\nst.session_state.uploaded_df          # Original data\nst.session_state.preprocessed_df      # Cleaned data\nst.session_state.encoders             # Fitted encoders\nst.session_state.scalers              # Fitted scalers\nst.session_state.models               # Trained models\nst.session_state.predictions          # Model predictions\n```\n\n---\n\n## 💻 Development Workflow\n\n### Adding a New Feature\n\n1. **Create a new module** (e.g., `my_feature.py`)\n\n```python\ndef run_my_feature(df):\n    \"\"\"Main feature function\"\"\"\n    st.markdown(\"### 🎯 My Feature\")\n    # Implementation\n```\n\n2. **Import in app.py**\n\n```python\nfrom my_feature import run_my_feature\n```\n\n3. **Add to navigation** (in app.py)\n\n```python\nif selected == \"My Feature\":\n    run_my_feature(st.session_state.uploaded_df)\n```\n\n4. **Test and debug**\n\n```bash\nstreamlit run app.py\n```\n\n5. **Commit and push**\n\n```bash\ngit add .\ngit commit -m \"Add my feature\"\ngit push origin main\n```\n\n### Modifying Configuration\n\nEdit `config.py` for:\n- Colors and theme\n- Algorithm parameters\n- File size limits\n- Feature flags\n\n---\n\n## 🧪 Testing\n\n### Running Tests\n\n```bash\n# Run all tests\npytest\n\n# Run with coverage\npytest --cov=.\n\n# Run specific test file\npytest tests/test_preprocessing.py\n```\n\n### Testing Modules\n\n**Example test structure:**\n\n```python\n# tests/test_preprocessing.py\nimport pandas as pd\nfrom preprocessing import run_preprocessing\n\ndef test_preprocessing():\n    df = pd.DataFrame({'A': [1, 2, 3]})\n    # or venv\\Scripts\\activate on Windows\n```\n\n### Issue: TensorFlow not found\n\n**Solution:** Install explicitly or check GPU drivers.\n\n```bash\npip install tensorflow\n```\n\n### Issue: Streamlit caching issues\n\n**Solution:** Clear cache and restart.\n\n```bash\nstreamlit cache clear\nstreamlit run app.py\n```\n\n### Issue: Large file processing is slow\n\n**Solution:** Use chunking or data sampling.\n\n```python\n# Sample data for testing\ndf_sample = df.sample(n=10000, random_state=42)\n```\n\n---\n\n## 📚 Resources\n\n- [Streamlit Documentation](https://docs.streamlit.io/)\n- [Scikit-learn Guide](https://scikit-learn.org/stable/)\n- [TensorFlow Docs](https://www.tensorflow.org/)\n- [Pandas Reference](https://pandas.pydata.org/docs/)\n\n---\n\n## 🤝 Contributing\n\nFor major changes:\n\n1.
